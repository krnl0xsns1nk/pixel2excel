# OCR Project - Image to Text

تحويل صورة إلى نص باستخدام Tesseract.js - معالجة محلية 100%

## Setup

```bash
# Install dependencies
pnpm install

# Run development server
pnpm dev

# Build for production
pnpm build
```

## Usage

1. افتح المتصفح على http://localhost:3000
2. اختر صورة تحتوي على نص (عربي أو إنجليزي)
3. انتظر المعالجة
4. النص المستخرج سيظهر في الـ textarea

## Tech Stack

- TypeScript
- Webpack
- Tesseract.js (Arabic + English)
- Pure HTML (no styling)
